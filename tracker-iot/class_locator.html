◆ watchDog()

byte Locator::watchDog ( )

Position algorithm.

This method decide if the position has changed or not and if the position hasn't be send for too long

Returns
Return a byte, if the first bit (LSB) is true, the position need to be send, if the 2nd bit is true, the tracker is outside the safezone
Precondition
Before using this method, please initialize the class
Bug:
The decision algorithm seems to be buggy, see dedicated github issue

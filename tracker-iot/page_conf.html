Configuring the tracker

In this page you will find detailed instruction to configure the tracker. All tracker configuration is editable in config.h file.

GPRS Configuration

The tracker use the GPRS network to communicate with the server. GPRS is a network protocol for mobile communication, used in particular in our cell-phones.
To use this network, you should so have a phone plan and a SIM card. You also need to enter parameters in the tracker to log into the network.
These parameters are APN url/address and GPRS credentials are provided by your mobile operator.
Enter the APN url/address in APN array. GPRS user and password should be enter in GPRS_USER and GPRS_PSWD. Don't forget to enter your SIM pin in the SIM_PIN macro if your SIM card have one.

MQTT Configuration

The internet protocol used by the tracker to communicate with server is MQTT.
To work, MQTT needs somes informations :

  • Broker URL/Addr (BROKER)
  • TCP/IP Port used (MQTT_PORT)
  • Transceiver topic (TOPIX_TX)
  • Receiver topic (TOPIX_RX)
  • MQTT user & password (MQTT_USER, MQTT_PSWD)

The two first parameters are given when you have install your MQTT broker. The three last, are sent to you when you create a new tracker on the mobile application.

Miscellaneous parameters

Modules baud

If you have choice to not used the preconized GPS and/or GPRS modules, you may need to change the speed of the serial communcation. You can do it with GSM_BAUD and GPS_BAUD.

Tracker algorithm configuration

Triggered distance

You can edit the sensibility of the position change with DISTANCE_TRIG. If you increase it, the detection of a movement will be coarser.

Position Intervals

The tracker used some intervals to limit and/or force the sending of a position of the server.
These are the interval macros :

  • MINIMAL_INTERVAL
  • DEFAULT_INTERVAL
  • DEFAULT_INTERVAL_WHEN_ALM

Safezone

You can increase or decrease the diameter of the safezone with DEFAULT_SAFE_ZONE_DIAM. This diameter is used to know when the tracker is outside the autorized zone and so when the vehicle is theft.

Remarks
We advise you not to decrease the diameter of the safezone because of the GPS module accuracy.

USB Serial baud

The speed of the USB debug serial communication can be modifided with USB_BAUD.
